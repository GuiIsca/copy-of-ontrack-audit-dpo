# Planta Layout

- Visualização por loja da Planta da Loja (PDF) e Layout do Formato (PDF/DWG).
- Seleção de loja e pré-visualização (PDF) em iframe; DWG para download.
- Admin faz upload e mantém histórico; resumo mostra o mais recente por tipo.
- Disponível para todos os perfis (consulta); gestão é só para Admin.
